# AI Practitioner

AI学习与实践项目集，汇集AI领域的工具、资源和实践案例。

## 🚀 子项目

### AI Stars
发现AI行业最杰出人物的导航页面。

**功能特点：**
- 收录20+位AI行业大牛（CEO、研究员、创业者）
- 高级暗色模式设计，灵感来自Linear/Apple
- 实时搜索过滤
- 按公司筛选
- 响应式布局，支持移动端

**技术栈：** 纯HTML/CSS/JavaScript，无需后端

**部署：** 支持GitHub Pages、Vercel、Netlify

**访问：** [ai-stars/](ai-stars/)

**收录名单包括：**
| 分类 | 人物 |
|------|------|
| 图灵奖得主 | Geoffrey Hinton, Yann LeCun, Yoshua Bengio |
| AI公司CEO | Sam Altman (OpenAI), Jensen Huang (NVIDIA), Demis Hassabis (DeepMind) |
| 研究领袖 | Andrew Ng, Fei-Fei Li, Jeff Dean, Ilya Sutskever |
| 创业新锐 | Mustafa Suleyman, Aravind Srinivas, Alex Wang |

### cc-switch
